#### uvc-ctrl
This is small utility for set and display uvc camera control option (brightness, saturation etc.)

Current support control option:

1.    Brightness
2.    Contrast
3.    Hue
4.    Saturation
5.    Sharpness
6.    Gamma
7.    Gain
8.    Backlight Compensation
9.    Auto/Absolute Focus
10.   Absolute Zoom
